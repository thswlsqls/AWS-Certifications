# DVA-C02 — Developer Associate

> 개발 관점의 시험. SDK 사용, API 통합, 코드 작성과 배포에 초점을 둡니다.
> 강의자료: [[AWS Certified Developer Slides v44.pdf|DVA 슬라이드 v44 (913p)]]

## 도메인 비중

> 공부한 시간이 아니라 **출제 비중에 비례해** 점수를 얻습니다. 비중 큰 도메인의 빈칸부터 채우세요.

| 도메인 | 비중 | `domains` 속성 값 |
| --- | --- | --- |
| AWS 서비스를 활용한 개발 | **32%** | `dva/development` |
| 보안 | **26%** | `dva/security` |
| 배포 | **24%** | `dva/deployment` |
| 트러블슈팅 및 최적화 | **18%** | `dva/troubleshooting` |

진도 현황은 [[진도표.base|진도표]]의 **DVA** 뷰에서 확인합니다.
