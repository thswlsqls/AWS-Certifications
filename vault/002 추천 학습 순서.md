# 추천 학습 순서

세 강의 모두 Stephane Maarek 강의라 슬라이드 뼈대가 거의 같습니다. IAM → EC2 → 스토리지 → 고가용성 → 데이터베이스 → S3까지는 세 슬라이드가 같은 흐름이고, 그 뒤에 시험별 전용 주제가 붙습니다. 같은 서비스를 세 번 공부하지 않으려면 **SAA → DVA → CloudOps** 순서가 효율적입니다.

- **SAA 먼저** — [[003 SAA-C03|SAA-C03]] 슬라이드(879p)가 공통 서비스를 가장 넓게 다룹니다. 여기서 채운 노트가 나머지 두 시험의 바탕이 됩니다.
- **DVA 다음** — [[004 DVA-C02|DVA-C02]]는 공통 서비스의 개발 관점 섹션을 보강하고, 새로 배우는 것은 배포·CI/CD 계열 5개 주제뿐입니다.
- **CloudOps 마지막** — [[005 SOA-C03|SOA-C03]]은 새 서비스보다 이미 아는 서비스의 운영 깊이를 묻는 시험이라, 앞 두 시험을 마친 상태에서 부담이 가장 적습니다.

## 진행 규칙

- 한 노트의 해당 관점 섹션을 끝낼 때마다 `status`와 `confidence`를 갱신합니다. 빈칸은 [[진도표.base|진도표]]의 **미완료** 뷰에서 확인합니다.
- 연습문제에서 틀린 내용은 미루지 말고 그 자리에서 해당 노트의 "시험 함정" 섹션에 `#exam/trap/<주제>` 태그로 기록합니다.
- 각 시험 응시 전 마지막 한 주는 [[001 Home|Home]]의 "시험 직전 한 주 루틴"을 따릅니다.
